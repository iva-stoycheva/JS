# Arithmetic Operators

## What are operators?

Operators are symbols that perform operations on values or variables.

Example:

```javascript
5 + 3
```

Here:
- `5` and `3` are operands.
- `+` is the operator.

---

## Arithmetic Operators

### Addition (`+`)

Adds two values.

```javascript
console.log(5 + 3); // 8
```

---

### Subtraction (`-`)

Subtracts one value from another.

```javascript
console.log(10 - 4); // 6
```

---

### Multiplication (`*`)

Multiplies two values.

```javascript
console.log(6 * 4); // 24
```

---

### Division (`/`)

Divides one value by another.

```javascript
console.log(12 / 3); // 4
```

---

### Modulus (`%`)

Returns the remainder after division.

```javascript
console.log(10 % 3); // 1
```

---

### Exponentiation (`**`)

Raises a number to the power of another number.

```javascript
console.log(2 ** 3); // 8
```
